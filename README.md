# Projeto de Análise de Banco de Dados Northwind

## Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de **praticar e demonstrar habilidades em SQL Server (T-SQL)** utilizando o banco de dados **Northwind**, simulando cenários reais de negócio.

A proposta é explorar dados de **vendas, clientes, produtos, funcionários e estoque** para gerar **análises orientadas à tomada de decisões comerciais**.

---

## Objetivos do Projeto
- Demonstrar proficiência em SQL Server  
- Explorar o banco de dados Northwind de forma analítica  
- Resolver problemas de negócio reais com consultas SQL  
- Gerar insights acionáveis a partir de dados  
- Aplicar boas práticas de escrita e organização de queries  

---

## Tecnologias Utilizadas
- **SGBD:** Microsoft SQL Server  
- **Linguagem:** T-SQL  
- **Ferramentas:** SQL Server Management Studio (SSMS)  
- **Versionamento:** Git & GitHub  

---

## Banco de Dados Utilizado
**Northwind Database**  
Banco de dados clássico amplamente utilizado para aprendizado, que simula um sistema de comércio eletrônico com informações sobre:
- Clientes  
- Pedidos  
- Produtos  
- Funcionários  
- Categorias  
- Transportadoras  

---

##  Análises Realizadas

###  1. Análise de Vendas
- Faturamento mensal e anual  
- Tendência de vendas ao longo do tempo  
- Análise de sazonalidade  
- Top 10 produtos mais vendidos  

### 2. Análise de Estoque
- Produtos com estoque crítico  
- Produtos sem vendas  
- Valor total de estoque por categoria  
- Identificação de produtos obsoletos  

###  3. Análise de Clientes
- Clientes com maior volume de compras  
- Segmentação de clientes por valor  
- Padrões de compra  
- Análise geográfica de clientes  

###  4. Análise de Desempenho
- Ranking de funcionários por vendas  
- Funcionários com maior desempenho comercial  
- Eficiência de vendas por região  

---

- SELECT, WHERE, ORDER BY

- Funções de agregação (COUNT, SUM, AVG)  
- GROUP BY e HAVING  
- Subqueries  
- CTEs (Common Table Expressions)  
- Funções analíticas (RANK, ROW_NUMBER)  
- Manipulação de datas e textos
